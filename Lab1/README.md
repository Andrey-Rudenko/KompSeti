# Лабораторная работа №1

## Задание

Установить на компьютер Oracle VM VirtualBox, Wireshark, WinSCP, PuTTY. Создать виртуальную машину Ubuntu 22.04.4 Server и настроить её так, чтобы к ней был доступ по сети с основного компьютера и у самой машины был выход в интернет. Установить на машине веб-сервер Apache2, создать в нём тестовую страницу. Получить доступ к странице с веб-браузера основного компьютера. Перехватить программой Wireshark пакеты HTTP GET и HTTP Response при обмене браузера со своим веб-сервером. Сохранить результат перехвата и отправить его для проверки.

## Что использовалось

- Oracle VM VirtualBox — чтобы запустить виртуальную машину.
- Ubuntu 22.04.4 Server — система на виртуальной машине.
- Apache2 — веб-сервер, на нём тестовая страница.
- PuTTY — чтобы подключаться к машине по SSH.
- WinSCP — чтобы передавать файлы на машину.
- Wireshark — чтобы перехватывать пакеты.

## Как делал

1. Настроил сеть виртуальной машины через виртуальный адаптер VirtualBox (Host-only Adapter), чтобы машина была видна с основного компьютера и при этом имела выход в интернет.
2. Поставил на Ubuntu Server Apache2 и положил в `/var/www/html` тестовую страницу.
3. С основного компьютера открыл в браузере IP-адрес виртуальной машины — страница открылась.
4. Включил Wireshark и снова открыл страницу в браузере.
5. В перехвате нашёл пакет HTTP GET (запрос браузера) и ответ сервера HTTP 200 OK со страницей.
6. Сохранил перехват в файл .pcapng.
