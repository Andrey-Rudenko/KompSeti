# Лабораторная работа №2

## Задание

Развернуть на базе виртуальной машины с Ubuntu виртуальный веб-хостинг с PHP и MySQL. Создать shell-скрипт, который добавляет пользователя, устанавливает ему заданный пароль, создаёт ему домашнюю директорию, добавляет таблицу базы данных и пользователя базы данных MySQL с тем же именем и паролем, создаёт виртуальный хостинг вида имя_пользователя.ru, по которому доступно содержимое домашней директории. Создать shell-скрипт, который удаляет пользователя по имени.

## Что использовалось

- Ubuntu Server — система на виртуальной машине.
- Apache2 — веб-сервер, виртуальные хосты.
- PHP — для тестовой страницы пользователя.
- MySQL — база данных пользователя.
- Bash — скрипты добавления и удаления пользователя.

## Скрипты

- [`add_user.sh`](./add_user.sh) — добавляет пользователя.
- [`del_user.sh`](./del_user.sh) — удаляет пользователя.

### add_user.sh

Запуск: `sudo ./add_user.sh <имя_пользователя> <пароль>`

Что делает:

1. Создаёт системного пользователя с домашней директорией `/var/www/<имя_пользователя>` и ставит ему пароль.
2. Создаёт папку `public_html` и в ней тестовую страницу `index.php`.
3. Создаёт в MySQL базу данных с именем пользователя, пользователя MySQL с тем же именем и паролем, выдаёт ему права на эту базу, создаёт в базе таблицу `users`.
4. Создаёт конфиг виртуального хоста Apache `<имя_пользователя>.ru`, где `DocumentRoot` смотрит на `public_html` пользователя, включает сайт и перезагружает Apache.

### del_user.sh

Запуск: `sudo ./del_user.sh <имя_пользователя>`

Что делает: отключает и удаляет виртуальный хост, удаляет домашнюю директорию, удаляет базу данных и пользователя MySQL, удаляет системного пользователя и его группу.

## Результат

После запуска `add_user.sh ivan pass123` сайт пользователя доступен по адресу `http://ivan.ru` (при настроенном DNS/hosts на машине, с которой открывают браузер), показывает страницу `index.php`, а в MySQL есть база `ivan` с таблицей `users` и пользователем `ivan`.
